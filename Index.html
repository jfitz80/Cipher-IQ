Cipher IQ

Crack the code. Train your logic.

Cipher IQ is a number-code logic puzzle game where players crack hidden ciphers using deduction, limited attempts, daily streaks, and feedback clues.

How to Play

Guess the hidden number code before your attempts run out. Feedback helps you determine which digits are exact matches and which digits are correct but misplaced.

Daily Cipher

A new Daily Cipher resets each day. Crack the code to build your streak and keep your logic sharp.
